### 介绍

相信玩过探探的人都知道，只有当两个人都同时右滑对方，才能够匹配到聊天，如果对方右滑了你，但是你却没有右滑到TA，那你们岂不是错过。如果不想错过，那么唯一的方法就是不断的右滑，右滑，右滑，但是持续的右滑，时间久了，手指会酸痛，你是否想过有这么一个工具，你只需要打开探探，然后就可以自动右滑，不需要动手指，过一段时间将会有上百个匹配呢？现在，你想要的，**探探快滑**可以满足你。
